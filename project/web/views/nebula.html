Specification

A modality for streamiling the construction of websites utilizing, HTML, CSS, Javascript, WebGL and more. Nebula is to be built of its own template engine and back-end system. Features including parsers, websockets, data management and security protocols.

    Triggers

    Triggers are eventhandlers that performs certain actions when activated. Triggers can be created using the syntax data-role="trigger". The type of trigger can be defined with data-type="some type". A target should be accompanied with almost all triggers. A target can be represented as the id tag of some other element.


    An example:
    
    <a data-role="trigger" data-type="modal" href="#a-modal">Link</a>

    <div id="a-modal" class="modal">
        <div class="modal-box">
            Modal content goes here.
        </div>
    </div>
  • May want to avoid href for security reasons?
  • Trigger and toggle may be the same or just similar.

    Modals are mini windows that display additional information or allow further dynamic interaction with certain elements. NebulaUI supports various modal styles and formats. There is the standard modal, slim modal and the dynamic modal.


    Standard Modal
    
    <a data-role="trigger" data-type="modal" href="#a-modal">Link</a>

    <div id="a-modal" class="modal">
        <div class="modal-box">
            Modal content goes here.
        </div>
    </div>
    Standard Modal Demo
    Slim Modal
    
    <a data-role="trigger" data-type="modal" href="#a-modal">Link</a>

    <div id="a-modal" class="slim modal">
        <div class="slim modal-box">
            Modal content goes here.
        </div>
    </div>
    Slim Modal Demo

    As you can see, the difference between the standard modal and slim modal is distinguished only by the use of the slim class addition. This is one of the goals of NebulaUI, to abstract customizable extensions as atomically possible.

  • May not need a separate modal for slim and standard.
    • Dynamic Modal

    Example Code
    
    <a class="bttn" data-role="trigger" data-type="modal" href="#a-modal">Dynamic Modal Demo</a>

    <div id="a-modal" class="modal">
        <div class="dynamic modal-box">
            <div class="menu" data-cluster="dynamic-modal-menu">
                <li data-role="trigger" data-type="modal-tab" href="#modal-tab-1">A</li>
                <li data-role="trigger" data-type="modal-tab" href="#modal-tab-2">B</li>
                <li data-role="trigger" data-type="modal-tab" href="#modal-tab-3">C</li>
                <li data-role="trigger" data-type="modal-tab" href="#modal-tab-4">D</li>
                <li data-role="trigger" data-type="modal-tab" href="#modal-tab-5">E</li>
            </div>
            <div data-role="stage" data-cluster="dynamic-modal-menu" class="content">
                <div id="modal-tab-1" data-role="modal-tab">This is modal tab A!</div>
                <div id="modal-tab-2" data-role="modal-tab">This is modal tab B!</div>
                <div id="modal-tab-3" data-role="modal-tab">This is modal tab C!</div>
                <div id="modal-tab-4" data-role="modal-tab">This is modal tab D!</div>
                <div id="modal-tab-5" data-role="modal-tab">This is modal tab E!</div>
            </div>
        </div>
    </div>
